# arXiv论文监控爬虫

一个基于Python的GUI应用程序，用于自动抓取和监控arXiv上特定领域的最新科研论文。

## 🚀 功能特点

- 图形化界面操作，简单易用
- 支持按学科分类筛选论文
- 自动保存历史记录到SQLite数据库
- 一键打开论文原文链接
- 实时显示爬取状态

## 🛠️ 技术栈

- Python 3
- Tkinter (GUI界面)
- Requests + BeautifulSoup4 (网络爬虫)
- SQLite3 (数据存储)

## 📦 安装和使用

1. 安装依赖：`pip install -r requirements.txt`
2. 运行程序：`python arxiv_crawler.py`

## 🎯 适用场景

- 科研人员跟踪最新文献
- 学生了解领域前沿动态
- 开发者学习Python爬虫和GUI编程
